What is this?

This is a site made by Evertras, an avid Heroes of the Storm fan! This site is intended to help share tips and ideas in dealing with various heroes in the game quickly and anonymously with as little fuss as possible.

How do I use it?

Click a hero. Add a tip. Upvote good advice. Downvote bad advice. That's about it! There's intentionally no login feature. You can upvote or downvote a single tip only once. If you want to see any new tips that have been added, check out the Newest Tips link at the top of the Heroes or Maps listings.

What happens after a big patch?

Votes are stored along with what patch they were given under. Once a new big patch comes out I'll be adding filters so you can see optionally filter out old patches. Open to suggestions on how exactly to handle this, check the links below for feedback channels.

Hey, it would be awesome if it did X!

Probably! Give me a shout at bfullj@gmail.com or check out the current Reddit thread. You can also help give input at Team Liquid! All constructive criticism is very welcome.

How can I help?

Vote! The whole system relies on people giving their input, so it's hardly going to be useful if we all don't chip in. Vote on stuff you know is good or bad advice, add your own, and everyone benefits. Besides using the above channels to offer valuable feedback, the next best thing you can do to help is to spread the word! The more people that use this site, the more awesome advice we can all give each other. Again, the more we all chip in, the better a resource this becomes!

Where'd you get the images?

The portraits for heroes were taken from HOTS Logs. The map graphics were taken from Battle.net.
