What is this?

This is a site made by Evertras, an avid Heroes of the Storm fan! This site is intended to help share tips and ideas in dealing with various heroes in the game quickly and anonymously with as little fuss as possible.

How do I use it?

Click a hero. Add a tip. Upvote good advice. Downvote bad advice. That's about it! There's intentionally no login feature. You can upvote or downvote a single tip only once.

What happens after a big patch?

Votes are stored along with what patch they were given under. Once a new big patch comes out I'll be adding filters so you can see optionally filter out old patches.

Hey, it would be awesome if it did X!

Probably! Give me a shout at bfullj@gmail.com or check out the current Reddit thread. You can also help give input at Team Liquid! All constructive criticism is very welcome.

How can I help?

Besides using the above channels to offer valuable feedback, the best thing you can do to help is to spread the word! The more people that use this site, the more awesome advice we can all give each other.
